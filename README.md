## Getting started

Clone the repo, then install the needed gems
```
$ bundle install --without production
```
Migrate the database
```
$ rails db:migrate
```
Run the test suite
```
$ rails test
```
If it passes, run in a local server
```
$ rails server
```
More information at [*RoR tutorial book*](http://www.railstutorial.org/book).
